# food-ism
*New to UIUC and curious about the restaurants on campus?* Welcome to food-ism, a website about my favorite food spots on campus with dish recommendations!


## Getting Started

### (Option 1) Run Locally
1. Install dependencies: `npm install`
2. Start the dev server: `npm start`
3. Open a browser and go to `http://localhost:8080/` to view your page.

### (Option 2) Launch Deployed Site
https://xupuomi.github.io/food-ism/
